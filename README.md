# qr-tracker-client

QR Code Tracker Frontend - React + TypeScript applications for admin dashboard and landing page.

## Structure
- `/admin` - Admin dashboard with authentication and analytics
- `/landing` - Public landing page

## Features
- JWT Authentication
- Location Management
- Scan Analytics Dashboard
- QR Code Link Generation
- Chart.js Visualizations

## Tech Stack
- React 18
- TypeScript
- Vite
- React Router
- Axios
- Chart.js

## Deployment
Both apps can be deployed separately on Vercel or Netlify.
